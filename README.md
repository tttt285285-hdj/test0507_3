# 比賽計時系統 (Competition Timer System)

這是一個基於 React 開發的比賽計時工具，專為需要雙方同時或分別計時的比賽場景設計（如辯論、棋類或體育競賽）。

## 核心功能

- **雙隊獨立控制**：提供 A 隊與 B 隊獨立的計時面板。
- **完整控制功能**：支援「開始」、「暫停」與「一鍵重設」。
- **響應式介面**：採用 Tailwind CSS 打造，在不同設備上都能保持良好的顯示效果。
- **直觀顯示**：使用清晰的時、分、秒格式顯示。

## 本地運行 (Run Locally)

**必要條件 (Prerequisites):** Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
